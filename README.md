## Aplicación de Diario
La aplicación Diario es una aplicación web poderosa e intuitiva que permite a los usuarios crear y administrar sus entradas de diario personales. Ya sea que esté buscando documentar sus pensamientos, emociones o experiencias, esta aplicación proporciona una plataforma segura y sin inconvenientes para sus necesidades de registro diario.

## características clave
- Autenticación de usuario: cree una nueva cuenta o inicie sesión en su cuenta existente para acceder a su espacio de diario personalizado.
- Cree y administre entradas: escriba y guarde sus entradas de diario sin esfuerzo. Edite, elimine y organice sus entradas con facilidad.
- Carga de imágenes: mejore las entradas de su diario cargando y adjuntando imágenes que complementen su contenido escrito.
- Persistencia de datos: las entradas de su diario se almacenan de forma segura, lo que le permite acceder y consultarlas cuando lo necesite.
- Interfaz intuitiva: disfrute de una interfaz limpia y fácil de usar diseñada para brindar una experiencia de registro agradable.
- Funcionalidad de cierre de sesión: cierre sesión de forma segura en su cuenta para proteger su privacidad y garantizar la seguridad de las entradas de su diario.

## Tecnologías
- Firebase: proporciona una infraestructura de back-end confiable y escalable para la autenticación de usuarios y el almacenamiento de datos.
- Redux Toolkit: facilita la gestión eficiente del estado, lo que garantiza una experiencia de usuario fluida y receptiva.
- SweetAlert2: mejore la interacción del usuario con hermosas alertas y notificaciones personalizables.
- Material UI: ofrece un amplio conjunto de componentes y estilos preconstruidos para crear una interfaz de usuario visualmente atractiva.
- Cloudinary: permite la carga y administración de imágenes sin problemas, lo que permite a los usuarios adjuntar imágenes a sus entradas de diario.
- API de Cloudinary: proporciona integración con la plataforma de Cloudinary, lo que permite una gestión y recuperación de imágenes eficientes.

## Empezando
1. Clona el repositorio.
2. Instale las dependencias requeridas usando npm install.
3. Configure su proyecto de Firebase y establezca las credenciales necesarias.
4. Inicie el servidor de desarrollo mediante npm start.
5. Acceda a la aplicación Journal en su navegador en http://localhost:3000.
